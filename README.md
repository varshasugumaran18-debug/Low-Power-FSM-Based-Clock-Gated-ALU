# Low-Power-FSM-Based-Clock-Gated-ALU

## Description
Designed and implemented both a conventional FSM-based ALU and a clock-gated FSM-based ALU in SystemVerilog HDL. The designs support arithmetic, logic, shift, multiplication/division, and advanced operations. A comparative analysis was performed to evaluate the reduction in switching activity and power consumption achieved through clock gating.

## Features
- 16-bit Modular ALU
- Conventional FSM-Based ALU
- Clock-Gated FSM-Based ALU
- Arithmetic and Logic Operations
- Shift and Rotate Operations
- Multiplication and Division Operations
- FPGA Implementation and Verification

## Tools Used
- SystemVerilog HDL
- ModelSim
- Intel Quartus Prime
- FPGA (DE10-Lite)

## Results
- Functional verification of both ALU architectures completed using ModelSim.
- Successfully implemented on FPGA.
- Comparative analysis shows that the clock-gated FSM-based ALU reduces switching activity and power consumption compared to the conventional FSM-based ALU.
